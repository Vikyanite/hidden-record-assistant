# LOL隐藏战绩查询小助手
## 前言

之前因为听说查战绩能提高单排胜率，于是就百度找到了隐藏战绩查询的网页（非常好用~）。

但是一直以来有几个痛点就是：

- 每次只能一个人一个人查
- 而且网站失灵时不灵的，最好是能多个网站一起查询
- 希望能过滤掉不是排位的战绩

于是这款帮助我上分的小助手就顺应而出啦~

## 主要功能

其实就是根据上面的痛点推出的功能啦~

- 可以一次性查询房间内的**己方**所有人
- 可以过滤掉不是排位的战绩
- 可以多个网站一起查询，从而保证能查到这个人的战绩
